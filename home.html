Welcome to Ethereum & Bitcoin Crypto Transaction Notifier!

A modern, open-source Chrome extension for real-time monitoring of your Ethereum and Bitcoin addresses.
Stay instantly informed about new transactions—securely and privately.

🚀 Key Features

  • 🔔 Real-Time Notifications: Get instant browser notifications for every new ETH or BTC transaction.
  • 🔒 Privacy-First: All data is stored locally. No tracking, no analytics, no data ever leaves your browser.
  • 🌐 Multi-Language Support: Use the extension in English, فارسی, العربية, Русский, 中文, and more.
  • 🛡️ Secure & Open Source: 100% open source, auditable, and built with security best practices.
  • ⚡ Easy Setup: Add your ETH/BTC addresses and (optionally) your Etherscan API key—done!

📝 Quick Start Guide

  1. Install the extension from the Chrome Web Store or load it as an unpacked extension.
  2. Open the extension popup and enter your Ethereum and/or Bitcoin addresses.
  3. For Ethereum, enter your Etherscan API key (get one for free from Etherscan).
  4. Enable monitoring for ETH and/or BTC.
  5. Receive instant notifications for every new transaction!
