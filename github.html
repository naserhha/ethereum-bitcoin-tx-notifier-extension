Project on GitHub

Ethereum & Bitcoin Crypto Transaction Notifier is an open-source Chrome extension designed for real-time monitoring of Ethereum and Bitcoin addresses. It provides instant, privacy-focused notifications for new transactions, with a modern, user-friendly interface and robust multi-language support.

  • 🔍 Transparent & Auditable: 100% open source. Review, audit, and verify every line of code for maximum trust.
  • ⚡ Real-Time Monitoring: Instantly notifies you of new ETH & BTC transactions using secure APIs (Etherscan, Blockstream).
  • 🌐 Multi-Language Support: Fully internationalized UI for global accessibility.
  • 🔒 Privacy-First: No tracking, no analytics, no data ever leaves your browser.
  • 🛠️ Easy Customization: Fork, extend, or contribute new features and translations.
  • 🤝 Community Contributions: Issues, pull requests, and feedback are welcome! See the repository for guidelines.
