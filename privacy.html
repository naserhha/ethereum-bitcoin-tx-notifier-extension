Privacy Policy

Your privacy is extremely important to us. This policy explains how the Ethereum & Bitcoin Crypto Transaction Notifier extension ("the Extension") handles your data and protects your privacy.

  • 1. No Data Sent to Servers: All your data—including wallet addresses, API keys, and transaction history—is stored locally in your browser. No information is ever sent to any external server or third party.
  • 2. No Tracking or Analytics: The Extension does not use any analytics, tracking scripts, or advertising networks. Your activity is never monitored, logged, or shared.
  • 3. API Key Privacy: Your Etherscan API key is only used locally to fetch your Ethereum transactions. It is never transmitted to any server except the official Etherscan API, and only from your own browser.
  • 4. User Control: You can clear all your data at any time from the Extension’s settings. You are always in control of your information.
  • 5. Security: All API requests are made over secure HTTPS connections. The Extension requests only the minimum permissions required to function.
  • 6. Open Source Transparency: The full source code is available on GitHub for review and audit by anyone.

If you have any questions or concerns about privacy, please contact the developer or open an issue on the GitHub repository.
